Datenschutz

Datenschutzerklärung dieser Website.

Diese Website dient der persönlichen und beruflichen Darstellung von Christoph Dietrich. Beim Besuch der Website werden durch mich keine eigenen Tracking-Tools, keine Analyse-Dienste, kein Newsletter und kein echtes Kontaktformular eingesetzt.

Hosting über GitHub Pages

Diese Website wird über GitHub Pages bereitgestellt. Beim Aufruf der Seite können technisch notwendige Zugriffsdaten verarbeitet werden, etwa IP-Adresse, Zeitpunkt des Zugriffs und aufgerufene Dateien.

Die Verarbeitung erfolgt durch den Hostinganbieter im Rahmen der Bereitstellung, Sicherheit und technischen Auslieferung der Website.

Schriftarten

Auf dieser Website werden Schriftarten von Google Fonts eingebunden. Beim Laden der Schriftarten kann eine Verbindung zu Servern von Google hergestellt werden. Dabei können technische Daten wie die IP-Adresse übertragen werden.

Eine lokale Einbindung der Schriftarten ist für eine spätere Version vorgesehen.

Cookies, Tracking und Formulare

Diese Website setzt aktuell keine eigenen Cookies, kein Tracking, keine Analyse-Tools und kein Kontaktformular ein.

Ein geschützter Kontaktweg wird vorbereitet. Sobald ein externer Dienst oder ein Formular eingebunden wird, wird diese Datenschutzerklärung entsprechend angepasst.

Bilder und lokale Dateien

Die auf der Website verwendeten Bilder und grafischen Elemente werden lokal über die Website-Dateien eingebunden.
